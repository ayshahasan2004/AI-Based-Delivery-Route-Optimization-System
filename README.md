# AI-Based Delivery Route Optimization System

This project applies Artificial Intelligence techniques to optimize package delivery routes for a local store.

The system assigns packages to vehicles while respecting weight capacity constraints.

It aims to minimize total travel distance while prioritizing urgent deliveries.

Two optimization algorithms are implemented: Genetic Algorithm (GA) and Simulated Annealing (SA).

The user selects the algorithm through an interactive interface.

Input data (packages and vehicles) is read from external files.

The system ensures all valid packages are assigned efficiently.

High-priority packages are delivered earlier when possible.

Edge cases such as overweight packages are handled properly.

The system compares performance in terms of execution time and solution quality.

GA provides better route optimization for large datasets.

SA offers faster execution suitable for real-time scenarios.

Test cases validate correctness, efficiency, and scalability.

This project demonstrates practical AI applications in logistics optimization.
